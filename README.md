# PowerBI
The Flight Booking & Travel Data Analysis Dashboard is an interactive Business Intelligence project developed using Power BI to analyze and visualize airline booking data. The dashboard provides a comprehensive view of key metrics such as ticket revenue, passenger demographics, travel patterns, and payment behaviors.

By transforming raw data into intuitive visual insights, the project helps identify trends across airlines, departure cities, travel classes, and customer segments. It enables users to understand how different factors—such as age groups, booking methods, and destinations—impact ticket pricing and overall revenue.

The dashboard is designed with interactive elements, allowing users to filter and explore data dynamically for deeper analysis. This makes it a powerful tool for uncovering actionable insights that can support strategic decision-making in the travel and aviation industry.

Overall, the project demonstrates the effective use of data visualization, data transformation, and analytical thinking to solve real-world business problems and extract meaningful insights from complex datasets.
